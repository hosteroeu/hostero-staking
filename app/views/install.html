Minimum

  • Internet connectivity
  • Operating System
    • Ubuntu: 16.04 or 18.04 (recommended)
    • Windows 10 64bit: Pro, Enterprise or Education (experimental)
  • Docker
    • 17.06.x-ee (recommended)
    • all supported versions (experimental)
  • 1 GiB Storage
  • 1 GiB RAM

Beginner guide

Hostero requires Docker to work. The installer will install Docker as well.

Install the Universal Miner

Run this command in the Terminal:

curl https://www.hostero.eu/install?a={{account.name || 'ACCOUNT_ID'}} | sh

Advanced guide

Hostero requires Docker to work. We recommend to install Docker via our script, but you can install it using alternative ways as well (supported versions).

1 Install Docker

Run this command in the Terminal:

Ubuntu 16.04

curl https://releases.rancher.com/install-docker/17.12.sh | sh

Ubuntu 18.04

curl https://releases.rancher.com/install-docker/18.06.sh | sh

 

2 Install Hostero

Run this command in the Terminal:

docker run -e CATTLE_HOST_LABELS="account={{account.name || 'ACCOUNT_ID'}}" --rm --privileged -v /var/run/docker.sock:/var/run/docker.sock -v /var/lib/rancher:/var/lib/rancher rancher/agent:v1.2.9 http://infra.hostero.eu:8080/v1/scripts/2E28C27EE87F5E1A6B7F:1514678400000:ShibdGaXFOlmN3QGGHj1UFuP1lU

 

Notes

If everything goes well and the device doesn't show up in Hostero after a couple of minutes, try running this command in the Terminal:

mkdir -p /etc/docker; echo '{"dns": ["8.8.8.8", "8.8.4.4"]}' > /etc/docker/daemon.json; service docker restart

Optional

Cloud providers have an option to set a startup script. The script will execute after the VPS is created, so you don't have to manually install it.

Add this script to user data:

#!/bin/bash
curl https://www.hostero.eu/install?a={{account.name || 'ACCOUNT_ID'}} | sh

Note: The commands are specific to your account, do not share them with others

Important: Replace ACCOUNT_ID with yours. You need to Login.
